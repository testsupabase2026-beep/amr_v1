# NAMAA Analytics Agent
# Pinned to the verified working runtime (env: qwen3-tts) for reproducible deploys.
#
# Embeddings run via the Google Gemini API (langchain-google-genai) — the model runs
# on Google's servers, so the heavy local-inference stack (torch, sentence-transformers,
# transformers) is NO LONGER a runtime dependency. This trims the Docker image by well
# over 1 GB. Requires GOOGLE_API_KEY at runtime.
#
# To run the LOCAL embedding fallback instead (no GOOGLE_API_KEY), reinstall the local
# stack:  pip install langchain-huggingface sentence-transformers torch
# (the FAISS index must then be rebuilt with the local 384-dim model — see
#  scripts/build_dwh_index.py).
